 Habit Tracker

A Flask-based habit tracker with dark mode, badges, and analytics to help users build consistent routines.

✨ Features
- 🌓 Dark mode support
- 🎖️ Badge system for milestones
- 📊 Analytics dashboard with charts
- 📝 Add, edit, and delete habits
